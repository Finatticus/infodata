什麼是結構型商品？

Fixed Coupon Note (FCN)

Fixed Coupon Note (FCN) 是一種固定配息的結構型商品，提供投資人比傳統定存更高的固定利息，並以連結標的（如個股或ETF）的表現作為條件。特色在於定期支付利息，但不保本，若連結標的價格在到期時跌破履約價格，投資人可能損失部分或全部本金。

Stepdown FCN

Stepdown FCN為一般FCN的延伸架構，提供同樣的固定收益機制，並設有逐期遞減的提前出場價格，每期最後一日觀察是否提前出場。適合預期市場震盪偏多、希望提高中後期出場機會、追求穩定配息且能承擔一定標的下跌風險的投資人。

Bullet FCN

Bullet FCN是一種固定配息的結構型商品，商品利息只在第一個配息日配發，所以越早提前出場，換算的年化配息率越高，適合在多頭市場時選擇的架構。

Daily Range Accrual Note (DRAN)

Daily Range Accrual Note (DRAN)是一種浮動配息的結構型商品架構，提供投資人高於FCN的浮動配息收益。配息依標的股票價格表現浮動，若任一連結標的價格跌破履約價格 (Strike)，當天配息將不計，並有觸發自動提前出場機制 (KO) 即到期以履約價 (Strike) 轉換股票的風險。
