📊 結構型商品專區

Fixed Coupon Note (FCN)

Fixed Coupon Note (FCN) 是一種固定配息的結構型商品，透過將債券與選擇權結合，提供投資人比傳統定存更高的固定利息，並以連結標的（如個股或ETF）的表現作為條件。其特色在於定期支付利息，但不保本，若連結標的價格在到期時跌破約定門檻，投資人可能損失部分或全部本金。

Stepdown FCN

Stepdown FCN是從FCN延伸出來的結構型金融商品，設有「每月遞減的提前出場價」，提高了在到期前提前出場並鎖定收益的機會。

Bullet FCN

Bullet FCN是一種固定配息的結構型商品，商品利息只在第一個配息日配發，所以越早提前出場，換算的年化配息率越高，適合在多頭市場時選擇的架構。

Daily Range Accrual Note (DRAN)

Daily Range Accrual (DRA)，中文:「每日區間計息結構型商品」，是一種結構型金融商品，其相對於FCN的高配息來自於連結標的的每日股價表現是否落在約定的「計息區間」內。若每日股價皆符合條件，投資人便可獲得計息；若股價未在區間內，則無法獲得該日的利息。
